# Agent Skills Dashboard

Internal product dashboard for [Agent Skills for Context Engineering](https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering).

## Overview

A three-column product dashboard where users can:
- Browse and filter skills by category
- Activate/deactivate skills with toggles
- Manage active skills priority order
- Monitor context budget usage
- View activity logs

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
